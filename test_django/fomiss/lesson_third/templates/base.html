Домашняя страница

Убедиться во всех преимуществах Django и Python можно лишь на деле. Чтобы использовать подобные технологии, необходимо убедиться в том, что ваш веб-ресурс отвечает определенным требованиям. К примеру, для установки «джанго» надо, чтобы хостинг имел UNIX-подобную систему. Чтобы Django нормально функционировал, необходимо обеспечить доступ к одному из веб-серверов: Ngix, Apache или Lighttpd. Не стоит забывать и о том, что хостинг для «джанго» обязательно должен иметь возможность для запуска скриптов. Для этой цели обычно используется планировщик задач Cron.
